## Installation

To get started with this project, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/react-portfolio.git
    ```

2. **Navigate to the project directory**:
    ```bash
    cd react-portfolio
    ```

3. **Install dependencies**:
    ```bash
    npm install
    ```
